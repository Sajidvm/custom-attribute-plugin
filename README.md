#Attribute Selector Slider (Product UI Component)

A modern slide-in attribute selector UI built for eCommerce product pages.

This component allows customers to quickly select attribute value types using a smooth side drawer interface instead of traditional dropdowns.

Designed for better UX, faster selection, and mobile-friendly shopping.
